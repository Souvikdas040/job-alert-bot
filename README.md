# Daily Fresher Job Alert Bot

Automated job alerts for fresher software engineers in India.

## Features
- LinkedIn + Wellfound + Naukri
- Fresher / 0–2 yrs filtering
- HTML email alerts
- Telegram notifications
- GitHub Actions (daily 9 AM IST)

## Setup
1. Add GitHub Secrets
2. Push repo
3. Enable Actions

Done ✅
